Module Logs_async_reporter

val reporter : unit -> Logs.reporter

The following reporter will play nice with Async's runtime, it will behave synchronously for the log functions of this module and asynchronously for those of the Logs module (see Logs.sync).

It reuses Logs_fmt.reporter and will produce colorful output if the standard formatters are setup to do so. For example it can be used instead of Logs_fmt.reporter in the full setup example.
